CSS Box Model

In CSS, the term "box model" is used when discussing about design and layout.
The CSS box model consists of; margins, borders, padding.
Padding: Padding is used to create space around an element's content, inside of any defined borders.
Border: The border allows us to specify the style, width and color of an element's border.
Margin: Margins are used to create space around elements, outside of any defined borders. It consists of top, bottom, left and right margins.
